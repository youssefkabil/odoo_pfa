# Gestion des Courriers (Courrier Management)

This is an Odoo module designed to manage incoming and outgoing mail (courriers).

## Features

- **Manage Mail Types**: Distinguish between Incoming (`Entrant`) and Outgoing (`Sortant`) mail.
- **Track Details**: Record Reference, Sender, Receiver, Subject, and Date.
- **Status Tracking**: Monitor the status of the mail through various stages:
  - Draft (`Brouillon`)
  - Sent (`Envoyé`)
  - Received (`Reçu`)
  - Archived (`Archivé`)
- **Description**: Add detailed notes or descriptions for each mail item.

## Technical Details

- **Model**: `courrier.management`
- **Dependencies**: `base` module
- **Author**: KABIL Youssef

## Installation

1.  Clone this repository into your Odoo addons directory.
2.  Update the Odoo app list.
3.  Install the "Gestion des Courriers" module.
